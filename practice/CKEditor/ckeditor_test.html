圖片

  • Installation
  • Inserting images

Image 是基本必須載入（plugins）的。
當 tool bar 只需要 upload（'uploadImage'）的功能時，需再額外載入 ImageUpload 或 ImageInsert。

請隨意試試。


但若想要使用 insert URL（'insertImage'）功能時，就必須載入 ImageInsert。
若載入 ImageUpload 則仍然只會有 upload 的功能；載入 ImageInsert 則同時包含 upload、insert URL 功能。

因此，其實也可以只載入 ImageInsert，tool bar 再設定是 'uploadImage'（只有 upload）還是 'insertImage'（upload 和 insert URL 都有）。

請隨意試試。


若只想要有 insert URL 功能的話，plugins 就要改用 ImageInsertViaUrl。

請隨意試試。


然而，上傳圖片會需要搭配 adapter 才能執行後續的實際上傳動作。
